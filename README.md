# Student Management System (SMS)

A modern, responsive web-based Student Management System that allows students to manage their academic information, courses, attendance, and grades.

## Features

- **User Authentication**
  - Student registration with email validation
  - Secure login system
  - Password management

- **Dashboard**
  - Overview of all academic activities
  - Quick access to all features
  - Real-time statistics

- **Course Management**
  - Add/Remove courses
  - View course details
  - Track course credits
  - Search and filter courses

- **Attendance Tracking**
  - Mark attendance for each course
  - View attendance history
  - Attendance statistics
  - Filter by date and status

- **Grade Management**
  - Record grades for assignments/exams
  - Calculate GPA
  - View grade history
  - Performance analytics

- **Profile Management**
  - View and edit personal information
  - Update contact details
  - Change password
  - Profile completeness tracking

## Technologies Used

- HTML5
- CSS3 (Modern styling with Flexbox/Grid)
- JavaScript (ES6+)
- LocalStorage for data persistence
- Responsive Design
